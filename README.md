adelaide's Node.js API.    
[Documentation](https://github.com/adelaide-Development/docs/tree/master/Atlas)    
# NOTE:  
Every class method must return a value, like so:  
```
class example {

    constructor() {}

    test(number, result) {
        if (result) {
            return result(number)
        } else {
            return console.log(number)
        }
    }

}
```
